### –ö–∞—Ä—Ä–∏—Ä–æ–≤–∞–Ω–∏–µ (Currying)
–ö–∞—Ä—Ä–∏—Ä–æ–≤–∞–Ω–∏–µ (–∫–∞—Ä—Ä–∏–Ω–≥) –∏–ª–∏ —á–∞—Å—Ç–∏—á–Ω–æ–µ –ø—Ä–∏–º–µ–Ω–µ–Ω–∏–µ —Ñ—É–Ω–∫—Ü–∏–∏ ‚Äî –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–µ —Ñ—É–Ω–∫—Ü–∏–∏ –æ—Ç –º–Ω–æ–≥–∏—Ö –∞—Ä–≥—É–º–µ–Ω—Ç–æ–≤ –≤ –Ω–∞–±–æ—Ä —Ñ—É–Ω–∫—Ü–∏–π,

```scala
/** –°–æ–∑–¥–∞—ë—Ç –Ω–æ–≤—ã–π —Å–ø–∏—Å–æ–∫ –Ω–∞ –æ—Å–Ω–æ–≤–µ —Å—Ç–∞—Ä–æ–≥–æ

  var filtered: List[Int] = List()        // –Ω–æ–≤—ã–π –ø—É—Å—Ç–æ–π —Å–ø–∏—Å–æ–∫
  for (element <- lst)                    // –ø–µ—Ä–µ–±–æ—Ä —ç–ª–µ–º–µ–Ω—Ç–æ–≤ —Å–ø–∏—Å–∫–∞ —Å –∑–∞–ø–∏—Å—å—é –∫–∞–∂–¥–æ–≥–æ –≤ element

      filtered = filtered.appended( element ) // –¥–æ–±–∞–≤–ª–µ–Ω–∏–µ —ç–ª–µ–º–µ–Ω—Ç–∞ –≤ –Ω–æ–≤—ã–π —Å–ø–∏—Å–æ–∫

  // –∞–Ω–∞–ª–æ–≥–∏—á–Ω–æ: def isOddMethod(x:Int)   =   x % 2 == 0


my_list_filter( isOdd,            List(1,2,3,4))          // -> List(2,4)
my_list_filter( (_:Int) % 2 == 0, List(1,2,3,4))          // -> List(2,4)
my_list_filter( _ % 2 == 0,       List(1,2,3,4))          // -> List(2,4)

// –∫–∞—Ä—Ä–∏—Ä–æ–≤–∞–Ω–∏–µ –º–µ—Ç–æ–¥–∞ my_list_filter, 
// —Ç–µ–ø–µ—Ä—å –æ–Ω –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç —Ñ—É–Ω–∫—Ü–∏—é, –∫–æ—Ç–æ—Ä–∞—è –¥–æ–ª–∂–Ω–∞ –ø—Ä–∏–Ω—è—Ç—å –µ–≥–æ –≤—Ç–æ—Ä–æ–π –ø–∞—Ä–∞–º–µ—Ç—Ä
```
